Hanging interfacec back to ToC or Class ToC

IRI: https://ouvroir.umontreal.ca/onto/display#HangingInterface

Hanging interface is a relation describing the hanging of an exhibit using an element. It allows qualifying the relationship of two or more things, where at least one is an exhibit (`display:Exhibit`)and at least one is an exhibition element (`display:Element`), indicating that this element is used in the hanging of the exhibit. Hanging interface can be the 'interface of' an exhibit and may be 'provided by' an exhibition element. In future versions of the Display Ontology, hanging interface could also receive additional qualifying properties, such as a type (using terms from a controlled vocabulary) or some other datatype properties like time span.
Is defined by
https://ouvroir.umontreal.ca/onto/display#
Term status
unstable
has super-classes
Interface c
is in domain of
interface of exhibit op

Pathway interfacec back to ToC or Class ToC

IRI: https://ouvroir.umontreal.ca/onto/display#PathwayInterface

Pathway interface is a relation describing the way of circulating between spaces. It allows qualifying the relationship of two or more exhibition spaces, indicating that these spaces are connected together so that one can go from one space to another. Pathway interface can be the 'interface of' an exhibition space and may be 'provided by' an exhibition element. In future versions of the Display Ontology, pathway interface could also receive additional qualifying properties, such as a type (using terms from a controlled vocabulary) or some other datatype properties.

has exhibitop back to ToC or Object Property ToC

IRI: https://ouvroir.umontreal.ca/onto/display#hasExhibit

Links an Exhibition space to an Exhibit that is spatially positioned in the exhibition space. The intended use of this relationship is not to be stated explicitly, but to be inferred from its relation with a building element. It will, for example, allow one to query for all the exhibits of an exhibition space given that they have a topological relation to elements of the building. This property and its sub-properties are defined in parallel with the `bot:hasElement` property and its sub-properties in order to apply the bot's core topological relationship patterns to the relation between Exhibition space and Exhibit.
Is defined by
https://ouvroir.umontreal.ca/onto/display#
